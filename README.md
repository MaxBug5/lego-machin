## 🛠️ ПОКРОКОВА ІНСТРУКЦІЯ: LEGO-РОБОТ НА WASD

### КРОК 1. Прошивка ESP32 Dev Module (`sketch_may18a.ino`)

1. Відключи від плати ESP32 **всі дроти**, які йдуть до Arduino Nano.
2. Підключи ESP32 до комп'ютера через USB-кабель.
3. Відкрий в Arduino IDE файл **`sketch_may18a.ino`**.
4. Переконайся, що в коді вказано твій Wi-Fi:
```cpp
const char* ssid = "Home WiFi";
const char* password = "Max_2243";

```


5. Зайди в меню **Tools ➡️ Board** (Інструменти ➡️ Плата) та обери **ESP32 Dev Module**.
6. Обери твій COM-порт у меню **Tools ➡️ Port**.
7. Натисни кнопку **Upload** (Стрілочка вправо).
8. Після успішного завантаження (`Done uploading`) відкрий **Tools ➡️ Serial Monitor**, вистав швидкість **115200**. Натисни кнопку *EN/RST* на платі. Вона виведе IP-адресу (наприклад, `119.168.1.50`). **Запиши або скопіюй її.**
9. Відключи USB-кабель від комп'ютера.

---

### КРОК 2. Прошивка Arduino Nano (`sketch_may18b.ino`)

1. Відключи від Arduino Nano **всі дроти**, крім ІК-діода на піні D4.
2. Підключи Nano до комп'ютера через USB-кабель.
3. Відкрий в Arduino IDE файл **`sketch_may18b.ino`**.
4. Зайди в меню **Tools ➡️ Board** та обери **Arduino Nano**.
5. **Важливо:** Зайди в меню **Tools ➡️ Processor** (Процесор). Якщо код не буде шитися, зміни `ATmega328P` на **`ATmega328P (Old Bootloader)`**.
6. Обери відповідний COM-порт у меню **Tools ➡️ Port**.
7. Натисни кнопку **Upload**.
8. Після появи напису `Done uploading` відключи USB-кабель від комп'ютера.

---

### КРОК 3. Складання апаратної частини (Залізо)

Коли обидві плати знеструмлені, з'єднай їх між собою за допомогою 5 дротів:

* **Живлення:** * Пін **`5V`** на ESP32 ➡️ Пін **`5V`** на Arduino Nano
* Пін **`GND`** на ESP32 ➡️ Пін **`GND`** на Arduino Nano


* **Шина даних SPI:**
* Пін **`GPIO 23`** на ESP32 ➡️ Пін **`D11`** на Arduino Nano
* Пін **`GPIO 18`** на ESP32 ➡️ Пін **`D13`** на Arduino Nano
* Пін **`GPIO 5`** на ESP32 ➡️ Пін **`D10`** на Arduino Nano


* **ІК-діод:** має залишатися підключеним до піна **`D4`** на Nano.

---

### КРОК 4. Запуск та Керування

1. Візьми свій USB-акумулятор (повербанк).
2. Підключи USB-кабель від повербанка **прямо в USB-роз'єм плати ESP32**. Обидві плати увімкнуться (на них загоряться світлодіоди).
3. Візьми телефон або ноутбук, підключи його до того самого домашнього Wi-Fi (`Home WiFi`).
4. Відкрий будь-який браузер (Chrome, Safari, Opera) і в адресному рядку введи IP-адресу, яку ти записав на Кроці 1 (наприклад: `192.168.1.50`).
5. Перед тобою з'явиться зелено-червоний пульт із сіткою кнопок WASD.
6. Направ ІК-діод на приймач LEGO машинки і тисни кнопки на екрані для керування!

### 🚨 ГОЛОВНЕ ПРАВИЛО БЕЗПЕКИ:

> Якщо тобі знову знадобиться змінити щось у коді через комп'ютер: **СПОЧАТКУ відключи провід `5V`, що з'єднує дві плати між собою**, і тільки потім вставляй USB-кабель у ПК! Не підключай дві плати до комп'ютера одночасно, якщо між ними кинуто міст живлення `5V`.
